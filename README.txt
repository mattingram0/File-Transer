*** README ***

Note:
- Socket connection uses port 9090
- [PATH] is path to /Networks and Distributed Systems Assignment/ directory

- To run the client program, navigate to /[PATH]/Networks/client/ and run:
:~ java Client

- To run the server program, navigate to /[PATH]/Networks/server/ and run:
:~ java Server

- To recompile client program, navigate to /[PATH]/Networks/client/ and run:
:~ javac Client.java InputException.java TransferException.java

- To recompile server program, navigate to /[PATH]/Networks/server/ and run:
:~ javac Server.java InputException.java TransferException.java ClientHelper.java
